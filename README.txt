学科内向け
コンパイルの仕方

make でコンパイルされます
make cleanでオブジェクトファイル等が消えます

遊び方
まずはシェルの窓を2つ用意してください
一方では
tail -f space.txt （ステージを描画します）
他方では
./a.out （入力を受け付け，現在の位置と速度を表示します）
を入力してください

Enterでゲーム開始です


"a" キーを押すと 星に引っ張られます
"s" キーを押すと 星と反発します
"space" キーを押すと 力がなくなります

うまく組み合わせて4回惑星を大きくしてください
R：ぶつかっちゃだめな星（自分よりおっきい）
B：ぶつかるべき星
O：自分よりちっちゃい星（ぶつかっても何も起きない）

衛星軌道について
Rの星の衛星軌道に入ることが可能です（Orbitalを遊んだことがある人なら分かるはず）
Rの星にうまいことかするように入ると衛星軌道に入ります
この時は、"a"キー"s"の力は衛星軌道になった星としか発生しません

ステージの変更について

ステージはrsc/下の、planets.txtに構成が書いてあります

そのファイルの書き方は、一行で一つの星について、
質量　x座標　y座標 x速度 y速度
と書きます。一番最初の星が自機になります。
自由にステージがかえられるので自分で色々作ってみてください
